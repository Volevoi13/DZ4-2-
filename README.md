## Задание 2: ##

Построение графика гармонического колебания.

## Назначение программы

Программа предназначена для построения графика гармонического колебания по введённым параметрам

На основе этих данных рассчитывается и строится график смещения во времени.

## Используемые физические формулы

Формула гармонического колебания:
  
$$
x(t) = A \cos(2\pi f\, t + \varphi)
$$

## Параметры ввода и построение графика

**Параметры, вводимые пользователем:**
- **Амплитуда**
- **Частота**
- **Фаза**

**Построение графика:**
- **Ось X** представляет время.
- **Ось Y** представляет смещение.
